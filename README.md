# Zenith Zero: Idle Descent

**Zenith Zero** is a high-octane auto-RPG dungeon descender built with Flutter and the Flame engine. Command the Zenith Zero mech as it descends through infinite floors of neon-lit dungeons, evolving its arsenal and mastering elemental synergies.

## 🚀 Key Features

*   **Auto-Combat Excellence**: High-fidelity combat logic featuring 'Endless Seek' Sentinel Blades and Celestial Jian visuals.
*   **Deep Progression**: A robust Meta Shop where you spend Embers to unlock permanent boons and archetype-defining Keystones.
*   **Dynamic Arsenal**: Evolving skills with 5 tiers of power, visual flourishes, and unique archetypes (Nova, Meteor, Firewall, etc.).
*   **Master Command Suite**: A stealthy, persistent developer environment for rapid balancing and testing (unlocked via secret key).
*   **Immersive Audio**: Procedural soundscapes with persistent volume control (muted by default for a clean start).

## 🛠️ Tech Stack

*   **Engine**: [Flame](https://flame-engine.org/)
*   **Framework**: Flutter (Dart)
*   **State Management**: Provider
*   **Platform Support**: iOS, Android, macOS, Web, Windows, Linux
