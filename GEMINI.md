# Project Mandates

This project uses PixelLab for AI-powered pixel art generation.

## Assets
- [PixelLab Documentation](PIXELLAB.md)

## Source Control
- **Assets**: Do not commit assets (especially those included in `.gitignore`) during development to preserve git history.

## Development Workflow
- **Validation**: ALWAYS take a screenshot or screen recording of the game after making code changes to validate the visual output. This MUST be done using a phone (e.g., physical device, iOS Simulator, or Android Emulator) and **NOT** the macOS desktop application.
- **Developer Options**: A hidden developer suite is available for testing.
  - **Access Key**: `TWANGPRO`
  - **Trigger**: Tap the **Floor Badge** (top-left) **5 times** rapidly to prompt for the key.
  - **Behavior**: Entering the key **toggles** the developer mode on or off.
  - **Tools**: Includes God Mode, Kill All, Floor Jumping, Currency Grants, and Meta Maxing.
